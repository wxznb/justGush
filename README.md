## 如何运行

```
    现在本地安装好 node与npm 或者是 yarn，git

    配置好git:
    https://help.github.com/articles/connecting-to-github-with-ssh/
    或者是谷歌，百度

    使用git将项目克隆下来:
      git clone git@github.com:wxznb/justGush.git
    进入项目文件:
      cd justGush
    使用npm将下载相关依赖包:
      npm install ||
      yarn install
    使用npm运行项目
      npm run dev ||
      yarn dev
```
